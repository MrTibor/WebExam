# Movie Explorer WebExam Project

How to run the project:

1. Download or unzip the project folder.

2. Open the project folder in Visual Studio Code or another code editor.

3. Open a terminal in the main project folder.

4. Install the required dependencies by running:

npm install

5. Make sure the `.env` file exists in the main project folder.

The `.env` file should contain:

OMDB_API_KEY=362e63da
PORT=3000

6. Start the server by running:

npm start

7. Open the website in a browser:

http://localhost:3000
